*olive-crt.txt*  A Game Boy-inspired olive colorscheme for Neovim

INTRODUCTION                                                    *olive-crt*

olive-crt.nvim is a Neovim colorscheme built from a Game Boy CRT style olive
palette, expanded with muted earthy accents for syntax clarity.

USAGE                                                           *olive-crt-usage*

  :set background=dark
  :colorscheme olive-crt

  :set background=light
  :colorscheme olive-crt

LUA SETUP                                                       *olive-crt-setup*

  require('olive_crt').setup({
    transparent = false,
    styles = {
      comments = {},
      keywords = { bold = true },
      functions = {},
      strings = {},
      variables = {},
    },
    plugins = {
      telescope = true,
      gitsigns = true,
      which_key = true,
      cmp = true,
      neo_tree = true,
      oil = true,
      notify = true,
      mini = true,
      treesitter_context = true,
    },
    overrides = {},
  })

  vim.cmd.colorscheme('olive-crt')

COMPATIBILITY                                                   *olive-crt-compat*

- Neovim includes Treesitter, diagnostics, semantic tokens, and plugin groups.
